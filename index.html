Project:Pediatric Pneumonia Recognition in Chest X-rays Using Deep Learning Techniques.

View More

1. Created a pneumonia detection system using DenseNet-161 on a Windows 8/10 PC with GPU acceleration and Python (Python IDLE 11). Used Google Colab for collaborative development.


2. Used a dataset of more than 5,000 medical pictures. In this 1,349 normal images and 3,883 pneumonia images.


3. Split dataset into 3,662 training images and 1,570 validation images.


4. Model architecture included DenseNet-161 with 3 transactions, each with 24 dense layers.


5. Achieved training results after 15 epochs: train_loss 0.1202, train_accuracy 0.9579, val_loss 0.1432, val_accuracy 0.9535.


6. Tested initial accuracy of 0.8862, improved to 0.95; employed visualization for loss and accuracy metrics, showcasing practical clinical applications in medical image analysis.
